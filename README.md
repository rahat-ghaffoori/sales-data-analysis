# Sales Data Analysis Project

## 📊 Overview
This project performs exploratory data analysis (EDA) on a retail sales dataset to uncover trends, customer behaviour, and key revenue drivers.

## 🎯 Objectives
- Analyse sales performance across regions, categories, and customer segments
- Identify key factors driving revenue
- Provide actionable business insights and recommendations

## 🛠️ Tools & Technologies
- Python (Pandas)
- Data Visualisation (Matplotlib)
- Jupyter Notebook

## 📁 Dataset
- Source: Kaggle Superstore Sales Dataset
- Includes order, customer, product, and regional sales information

## 🔍 Key Analysis Performed
- Data cleaning and preprocessing
- Feature engineering (time-based features)
- Sales trend analysis (yearly and monthly)
- Customer segmentation analysis
- Product and category performance evaluation

## 📈 Key Insights
- Sales show a steady upward trend over time
- Revenue is concentrated in specific regions and customer segments
- A small number of products drive a large share of total sales
- Seasonal patterns affect monthly performance

## 💡 Business Recommendations
- Focus on high-performing regions and customer segments
- Promote top-selling products to maximise revenue
- Improve underperforming categories
- Use seasonal insights for planning and forecasting

## 🚀 Outcome
This project demonstrates the ability to perform end-to-end data analysis and generate actionable business insights using real-world data.
