# Sales Data Analysis Project

## 📊 Overview
This project performs exploratory data analysis (EDA) on a retail sales dataset to uncover trends, patterns, and business insights.

## 🎯 Objectives
- Understand sales performance across different regions and categories
- Identify key factors affecting profit and revenue
- Explore customer purchasing behavior

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

## 📁 Dataset
- Source: Kaggle Superstore Sales Dataset
- Contains information on orders, sales, profit, customers, and regions

## 🔍 Key Steps Performed
- Data loading and inspection
- Handling missing values
- Data type analysis
- Initial statistical summary

## 🚀 Next Steps
- Data cleaning and preprocessing
- Data visualization
- Building an interactive dashboard (Power BI)
- Generating business insights
